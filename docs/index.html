University Projects

Emerging IT Recommendation Report Group Work

In this project, my group and I wrote 2 different recommendation reports for a company in need of a Content Management System, and another company in need of a Motion Capture Systems. Each recommendation report reviewed 3-4 different viable providers, and scored them based on many different requirements in accordance with the IEEE Standards for IT Adoption.

Software Engineering Web Application: Medical Record System

This was my first big solo project at UHCL. I created a Medical Record System using ASP.NET and a MySQL database. Deliverables included a scope and task list, requirements, architecture, design, and pseudo code documents.

CIS Senior Project: Django Inventory Management System

This was my senior project, in which 5 of my peers and I worked together to create an Inventory Management System for food pantries. I learned how to create the frontend and backend of the Django web application on my own. I also coordinated with both the database team and the report generator team in order to ensure connectivity to the web app. The project was a success, and I got a lot of experience working with other people and handling conflicts.
